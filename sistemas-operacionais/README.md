markdown# 🖥️ Aula: Sistemas Operacionais (Linux, Windows e macOS)

## 1. O que é um Sistema Operacional (SO)?
O SO é o software que gerencia o hardware do computador e fornece serviços para os programas aplicativos. Ele atua como o intermediário entre o usuário e o hardware.

---

## 2. Comparativo de Sistemas Operacionais


| Característica | 🐧 Linux | 🪟 Windows | 🍎 macOS |
| :--- | :--- | :--- | :--- |
| **Núcleo (Kernel)** | Monolítico (Linux) | Híbrido (NT) | XNU (Baseado em Unix/BSD) |
| **Código** | Aberto (Open Source) | Fechado (Proprietário) | Fechado (Proprietário) |
| **Licença** | Gratuito (na maioria) | Pago | Pago (Incluso no Hardware) |
| **Público-Alvo** | Servidores, Devs, IoT | Usuários domésticos, Gamers | Design, Devs, Executivos |

---

## 3. 🐧 Linux: O Sistema Aberto
O Linux não é um sistema único, mas um núcleo usado por várias **Distribuições (Distros)**.
*   **Vantagens:** Estabilidade, segurança, alto nível de personalização e gratuidade.
*   **Interface:** CLI (Terminal) é a força principal, mas possui interfaces gráficas (GNOME, KDE).
*   **Exemplos:** Ubuntu, Debian, CentOS, Fedora.

---

## 4. 🪟 Microsoft Windows: O Padrão de Mercado
Dominante no mercado corporativo e doméstico.
*   **Vantagens:** Compatibilidade de software (especialmente jogos), facilidade de uso e suporte de drivers.
*   **Arquitetura:** Baseada no kernel **Windows NT**. Utiliza o sistema de arquivos **NTFS**.
*   **Versões Atuais:** Windows 10 e Windows 11.

---

## 5. 🍎 macOS: O Ecossistema Apple
Exclusivo para computadores da Apple (MacBook, iMac).
*   **Vantagens:** Integração perfeita com hardware, interface fluida (UI/UX) e base Unix (excelente para desenvolvedores).
*   **Arquitetura:** Construído sobre o sistema **Darwin**, o que o torna muito estável e seguro.
*   **Ecossistema:** Sincronia nativa com iPhone, iPad e iCloud.

---

## 6. Comandos Básicos de Terminal (CLI)


| Ação | Linux / macOS | Windows (CMD) |
| :--- | :--- | :--- |
| Listar arquivos | `ls` | `dir` |
| Mudar de pasta | `cd` | `cd` |
| Criar pasta | `mkdir` | `mkdir` |
| Limpar tela | `clear` | `cls` |
